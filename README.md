Overview

This project is a small “Test Assessment” web application that demonstrates an employee/admin login flow and a basic assessment flow. It has:

Employee login to submit an answer to a sample question.

Admin login to view submitted answers.

A styled single-page front end that can also act as a static prototype.    ​

Tech Stack:
-Python 3
-Flask (web framework)
-SQLite (local database testassessment.db)
-HTML, CSS, JavaScript for the UI

Usage:
Open the app in the browser.

Click “Login” and use:
Employee: emp123 / emp123 → submit an answer.
Admin: ad123 / ad123 → view all answers.
